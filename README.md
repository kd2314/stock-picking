# stock-picking
small algorithm developed by km chen for picking stocks

## 0722-V1.1
1. 可以验证历史选股策略，后N天的涨幅
2. 新增每日准确率统计，验证策略正确性

## Usage
suitable for all operation systems(windows/linux/macos)
1. conda create -n your_env python=3.9
2. pip install -r requirements.txt
3. python app.py
